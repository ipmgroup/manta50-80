# manta50-80
Motor-Controller, FOC

### Manta50 key characteristics

    DC-Input 4.4V to 45V 
    Maximum DC / AC (per phase) current - 20A
    Texas Instruments C2000 Piccolo TMS320F28027F
    CAN - MCP2515
    Three-Phase Smart Gate Driver - DRV8305
    one GPIO input for PWM(not supported in FW)

with good cooling in water, you can increase the AC current to 80A. Replacing the resistance from 0.007R to 0.002R.
